# Global Tokens

<cds-alert status="danger" closable="false">

</cds-alert>
<br />

Design Tokens are global variables to configure the foundation of the
design system. Tokens control values such as color, spacing and
typography. Changes to a design token will propagate throughout the entire system.

## Installation

To use the layout system import the global CSS.

```scss
// import into sass or css
@import '~@clr/core/module.tokens.min';

// or import in html
<link href="/node_modules/@clr/core/module.tokens.min.css" rel="stylesheet">
```

## Usage

Tokens can be adjusted by setting the values on the `:root` selector.

```css
:root {
  --cds-token-typography-body-color: rgb(26, 26, 26);
}
```
