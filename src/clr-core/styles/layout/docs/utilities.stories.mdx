# Utilities

Outside of the horizontal, vertical and grid layout types the `cds-layout`
system provides additional utilities for managing space and visibility.

<cds-alert status="danger" closable="false">
  <cds-alert-content>This is a experimental API and will change. Do not use this in production enviroments!</cds-alert-content>
</cds-alert>
<br />

## Spacing / Pad

The `pad` attribute allows you to apply a pad space value that cooresponds to
the Clarity Design spacing token values.

## Display

The display option allows us to change the visibility of a element based on the
Clarity Design size breakpoint tokens.
