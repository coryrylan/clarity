# Layout

<cds-alert status="danger" closable="false">
  <cds-alert-content>This is a experimental API and will change. Do not use this in production enviroments!</cds-alert-content>
</cds-alert>
<br />

Clarity Core components do not provide surrounding white space. Spacing
between components is managed entirely by layout components. This ensures that
the system is as composable as possible while keeping white space predictable
and consistent.

The `cds-layout` component include three layout types. Each is designed to
handle particular layout senarios.

1. Horizontal
2. Vertical
3. Grid

## Installation

To use the layout system import the global CSS as well as the optional Web component
for improved IDE experience.

```scss
// import into sass or css
@import '~@clr/core/module.layout.min';

// or import in html
<link href="/node_modules/@clr/core/module.layout.min.css" rel="stylesheet">

// optional import in JavaScript
import '@clr/core/layout';
```

## Horizontal

The horizontal layout type allows horizontal rows of items that can wrap automatically.
Spacing, alignments and gap between items can all be configured with the component.

<Story id="experimental-layout-all--horizontal-layout" />

```html
<cds-layout type="horizontal" gap="sm">
  <cds-placeholder>1</cds-placeholder>
  <cds-placeholder>2</cds-placeholder>
  <cds-placeholder>3</cds-placeholder>
</cds-layout>
```

## Vertical

The vertical layout type enables stacks of items. This is usefull for spacing
cards, content and other block level items. Spacing, alignments and gap between
stacked items can all be configured with the component.

<Story id="experimental-layout-all--vertical-layout" />

```html
<cds-layout type="vertical" gap="sm">
  <cds-placeholder>1</cds-placeholder>
  <cds-placeholder>2</cds-placeholder>
  <cds-placeholder>3</cds-placeholder>
</cds-layout>
```

## Grid

The responsive grid layout type enables columns to be defined for layout. This is usefull
for controling the spacing of items within a container as well as responsive
layout. Spacing, alignments and gap between items can all be configured with
the component.

<Story id="experimental-layout-all--grid-layout-columns-responsive" />

```html
<cds-layout type="grid" cols="6@sm 3@md" gap="sm">
  <cds-placeholder>1</cds-placeholder>
  <cds-placeholder>2</cds-placeholder>
  <cds-placeholder>3</cds-placeholder>
  <cds-placeholder>4</cds-placeholder>
</cds-layout>
```
