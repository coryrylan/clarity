# Horizontal

<cds-alert status="danger" closable="false">
  <cds-alert-content>This is a experimental API and will change. Do not use this in production enviroments!</cds-alert-content>
</cds-alert>
<br />

The horizontal layout type allows horizontal rows of items that can wrap automatically.
Spacing, alignments and gap between items can all be configured with the component.

## Horizontal Layout

Horizontal layout is usefull for horizontal lists of items such as buttons,
tags or labels.

<Preview>
  <Story id="experimental-layout-all--horizontal-layout" />
</Preview>

## Wrap

By default the horizontal layout will allow items to wrap and ensure the gap
is only applied between layout items.

<Preview>
  <Story id="experimental-layout-all--horizontal-layout-wrap" />
</Preview>

## No Wrap

You can disable wrapping using the `wrap="none"` attribute. This will force
the items to follow standard CSS overflow rules (`hidden`/`scroll`).

<Preview>
  <Story id="experimental-layout-all--horizontal-layout-no-wrap" />
</Preview>

Alignments are relative to the size of the `cds-layout` container.
