# Grid Column Resize

<cds-alert-group status="warning" cds-layout="m-b:lg">
  <cds-alert>
    This component or utility is offered as a preview. This means we are currently working on it and seeking feedback.
    Please be aware that this component or utility may have breaking changes before we finish working on it.

Grid Columns by default provide a resize handle to enable a user to resize the column width.
Column resizing can be disabled by setting the `resizable` property to false.

<cds-alert status="info">To resize a column with a keyboard, focus the resize handle and use the left and right arrows to adjust the column.</cds-alert>
